# lambdas-golang

## Usage

To get a list of available targets, run the following command inside a component directory.

```bash
mage
```

## Dependencies

- [go](https://golang.org/)
- [mage](https://magefile.org/)
- [tofu](https://opentofu.org/)
- [aws-cli](https://aws.amazon.com/cli/)

### MacOS/Linux

```bash
brew install go
brew install mage
brew install --cask tofu
brew install awscli
```
